# Elce

Elce, Turk Isaret Dili ogrenmek isteyen kullanicilar icin gunluk dersler, kamera destekli pratikler, hiz testi ve senaryo tabanli egitimler sunan oyunlastirilmis bir mikro-ogrenme uygulamasidir.

## Sosyal Etki

Elce, isitme engelli bireylerin gunluk hayatta yasadigi iletisim bariyerini azaltmak icin toplumdaki bireylerin temel Turk Isaret Dili becerilerini gelistirmeyi hedefler.

## MVP Ozellikleri

- Gunluk kisa dersler
- XP ve seviye sistemi
- Streak sistemi
- Sinirsiz can
- Kamera destekli pratik
- Elce Kocu geri bildirimi
- Elce Hiz Testi
- SPM skoru
- Acil Isaretler bolumu
- Basit tekrar havuzu
- Senaryo tabanli pratik

## Teknik Yapi

- Frontend: React + Vite
- Kamera / el algilama: MediaPipe Hand Landmarker
- AI / model denemeleri: Python
- Veri saklama: localStorage
- Backend: MVP icin yok

## MVP Kapsami

Bu MVP tam otomatik Turk Isaret Dili cevirisi yapmaz. Belirli TID isaretleri icin kamera destekli pratik dogrulama sunar.
