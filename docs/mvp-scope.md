# MVP Scope

## MVP'de Olacaklar

- Ana sayfa
- Gunluk hedef
- XP sistemi
- Streak sistemi
- Sinirsiz can
- 3-5 temel Turk Isaret Dili dersi
- Quiz mekanikleri
- Kamera destekli pratik
- Elce Kocu geri bildirimi
- Elce Hiz Testi
- SPM skoru
- Acil Isaretler bolumu
- Basit tekrar havuzu

## MVP'de Olmayacaklar

- Tam otomatik Turk Isaret Dili cevirisi
- Gercek zamanli cumle cevirisi
- Kusursuz AI modeli
- Native mobil uygulama
- Cok dilli isaret dili destegi
- Gelismis rozet ve lig sistemi
- Buyuk database sistemi
- Kullanici giris/kayit sistemi

## Teknik Sinir

MVP, belirli isaretler icin kamera destekli dogrulama yapar. Model, kullanicinin el hareketlerini hedef isaretlerle karsilastirarak skor verir.
